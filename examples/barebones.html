Hyperledger Iroha v1

Initiated in 2016, first production release in 2019. Simple blockchain framework for enterprise solutions on C++.

Key Features

  • Written on C++ – fast and reliable, can be run on embedded devices
  • Linearly scalable CFT consensus
  • Various Client Libraries for frontend or mobile development
  • Unified builds
  • Tested pre-written commands and integration with HL Burrow EVM for smart-contracts – it is for you to choose
  • Integrated HL Explorer and HL Ursa

Hyperledger Iroha v2

Currently in development. Release planned for Q4 2020

Key Features

All the perks of v1 and also

  • Written in Rust – perfect blockchain language
  • Conditional triggers for different events
  • In-Memory Storage
  • Pluggable BFT consensus algorythm
  • DSL for smart-contracts
  • In the future - custom assets

Use-Cases

Iroha is a Universal enterprise blockchain and can be customised for many use-cases but there are some for which Iroha v2 would be just the perfect fit

  • Asset management
  • CBDC
  • Identity management
  • Record-keeping
  • Supply chains
  • Decentrilised exchanges
  • Decentrilised Economic Solutions
